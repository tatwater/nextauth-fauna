# nextauth-fauna
A Next.js app using NextAuth.js and its FaunaAdapter for authentication

## Setup

### Install packages

Run `% yarn install`.

### Create `.env.local` in project root:

```
FAUNA_ADMIN_KEY=""
FAUNA_CLIENT_SECRET=""

AUTH_SECRET=""
NEXTAUTH_URL="http://localhost:3000"

EMAIL_FROM=""
EMAIL_SERVER_HOST=""
EMAIL_SERVER_PASSWORD=""
EMAIL_SERVER_PORT=""
EMAIL_SERVER_USER=""

```
